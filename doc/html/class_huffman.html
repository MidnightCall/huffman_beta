Detailed Description

哈夫曼编码相关操作的类 用于记载符号和对应的二进制编码字符串； 主要用途： (1) 解码：读取根节点字段并且遍历树来抽取出所需要的信息 (2) 译码：调用getCode()函数来提取指定符号的二进制编码 从根节电到指定叶节点的路径决定了叶节点记录的符号的二进制编码， 往左走表示0，往右走表示1. (3) 构建哈夫曼树(私有方法，仅供内部方法调用) (4) 构建密码表(私有方法，仅供内部方法调用)
